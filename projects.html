M.Sc. Thesis · Deep Reinforcement Learning
Perfecting Tetris with Deep RL

MSc thesis project exploring whether model-free deep reinforcement learning can master Tetris. Compared value-based (DQN) against policy-based (PPO) approaches across multiple board sizes and state representations.

  • DQN achieved >2,000 lines per episode on 8×6 boards
  • PPO cleared max simultaneous lines on smaller boards
  • Explored 4-feature, 12-feature, full-board & "glimpse" state representations
  • Value-based methods showed superior generalization to larger boards

Atari Breakout — DQN Agent

A Deep Q-Network agent trained to play Atari Breakout from raw pixel input. Inspired by DeepMind's landmark paper, the agent learns to play from 260×160 RGB frames alone, with no game-specific features.

  • Surpassed average human score by 500%
  • Achieves near-perfect score on best runs
  • Trained entirely from raw pixel input (260×160 RGB)
  • CNN processes visual frames → Q-values for 4 actions

Information Retrieval · Data Structures
Inverted Index Search Engine

A search engine built over a JSON recipe dataset using an inverted index — the same core data structure powering Elasticsearch and major search engines. Enables fast full-text lookups across structured JSON data.

"pasta" → [doc_3, doc_7, doc_12, ...]
"chicken" → [doc_1, doc_4, doc_9, ...]
"garlic" → [doc_2, doc_4, doc_7, ...]
query("pasta AND garlic") → [doc_7]
  • Inverted index maps terms → document IDs for O(1) lookups
  • Supports boolean query expressions (AND/OR)
  • Applied to recipe JSON dataset for full-text ingredient search
